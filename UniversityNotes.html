EngSci Coursework Repository

** Disclaimer: Kindly note that this page is still under construction. As such some but not all of the embedded links function correctly.

Welcome to the EngSci Coursework Repository! In this page you will find the resources for the Engineering Science Program at the University of Toronto. The following is a compilation of typed notes, hand-written study notes, and solutions which will be continuously updated throughout university. Note that I do not claim to guarantee that everything is correct (in fact, I know for a fact that some of my solutions are not entirely correct) and I cannot guarantee completeness. I'm not paid to do this so I only put down what I think is necessary. Also acknowledge that these courses are revised each semester and are taught by different professors each term, so I would advise you to be cautious when using these resources. Additionally I have shared some of my final submissions and deliverables of lab reports, project reports, problem set solutions etc. Please use them as reference only and avoid plagiarism.
